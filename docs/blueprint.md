# **App Name**: QuizAI

## Core Features:

- Gmail Authentication: Secure user authentication using Gmail accounts with OTP verification.
- AI Quiz Generation: Generate quizzes on specified topics using the Gemini API based on user-defined parameters, functioning as a tool to create quizzes.
- Manual Quiz Creation: Enable users to manually create quizzes with a form interface for adding questions and answers.
- Quiz Saving and Management: Allow users to save generated or manually created quizzes to their profile.
- Quiz Download: Enable users to download quizzes in various formats (PDF, JSON, text).
- Quiz Playing and Result Tracking: Enable users to play saved quizzes and track their scores with a visual graphical representation of their performance.
- API Key Management: Allow users to enter and manage their Gemini API key, with a link to the Google AI Studio website for key retrieval.

## Style Guidelines:

- Primary color: A calm blue (#64B5F6), evoking a sense of trust and intelligence, reflecting the AI-driven nature of the quiz generation.
- Background color: A light blue (#E3F2FD) providing a clean and unobtrusive backdrop for the quizzes.
- Accent color: A vibrant orange (#FFB74D), drawing attention to key actions such as generating or saving quizzes.
- Body and headline font: 'Inter', a sans-serif font offering a modern, machined look that pairs well with both headlines and body text. 
- Use clean and consistent icons to represent different quiz actions and topics.
- Implement a clear and intuitive layout that prioritizes ease of use for both quiz generation and participation.
- Incorporate subtle animations to enhance user engagement, such as loading animations during AI quiz generation.
